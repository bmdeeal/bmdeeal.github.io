About

I'm Brenden Deeal, and I develop software for a variety of platforms. This site is a portfolio of several projects I've worked on.

My current major project is quig, a simple and friendly system for developing retro-style sprite scaling games in Lua, with extensive documentation and detailed examples.

Projects

Some of my projects include:

  • quig - a Lua-based environment for creating simple games with sprite scaling
  • ezjiv - an image viewer written in Java
  • ezpyle - a friendly text editor designed for access over limited connections (eg, serial)
  • curve-edit - a basic editor for POV-Ray curves written in JavaScript (in beta)
